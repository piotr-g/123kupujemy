# 123kupujemy.pl - Skup aut Śląsk

## Opis projektu
Profesjonalna strona internetowa dla firmy skupującej samochody w województwie śląskim.

## SEO Optymalizacje

### ✅ Zaimplementowane
- **Sitemap.xml** - mapa strony dla wyszukiwarek
- **Robots.txt** - instrukcje dla botów wyszukiwarek
- **Schema.org structured data** - dane strukturalne dla lepszej widoczności
  - LocalBusiness schema na stronie głównej
  - Article schema na stronach poradników
  - Breadcrumb schema dla nawigacji
- **Ulepszone meta opisy** - optymalne długości i słowa kluczowe
- **Canonical URLs** - unikanie duplikacji treści
- **Open Graph** i **Twitter Cards** - optymalizacja dla mediów społecznościowych
- **Alt attributy** dla wszystkich obrazów
- **Responsive design** - mobile-first approach
- **Structured navigation** - logiczna hierarchia stron
